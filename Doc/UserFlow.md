# User Flow: Urban Purifier

## 1. Entry Point
- 사용자가 웹사이트 접속 -> GitHub 로그인 또는 게스트 로그인.
- 저장된 게임이 있으면 '계속하기', 없으면 '새 도시 시작' 선택.

## 2. Main Gameplay Loop
1. **Explore**: 오염된 안개로 덮인 격자 중 하나를 클릭.
2. **Reveal**: 
   - 안전할 경우: 숫자가 표시되며 주변이 정화됨 (Flood Fill). 에너지 획득.
   - 오염 핵일 경우: HP 감소 및 주변 건물 피해.
3. **Build**: 획득한 에너지로 정화된 칸에 '주거 단지' 또는 '정화 타워' 건설.
4. **Expand**: 모든 안전 지대를 확보하거나 목표 에너지를 달성하면 다음 구역 해금.

## 3. Interaction Flow
[클릭] -> [서버 통신] -> [성공 애니메이션] -> [자원 UI 업데이트] -> [건설 메뉴 활성화]

## 4. Game Over / Save
- HP가 0이 되면 게임 오버 (복구 아이템 사용 가능).
- 창을 닫으면 GitHub Actions를 통해 배포된 백엔드가 DB에 상태 자동 저장.
